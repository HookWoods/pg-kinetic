# Support

## Documentation

Start with the published documentation at
[docs.pgkinetic.dev](https://docs.pgkinetic.dev). The installation,
configuration, operations, compatibility, and troubleshooting guides describe
the supported deployment paths.

## Getting Help

For a reproducible usage problem, open a GitHub issue and include:

- pg-kinetic version, Git commit, container tag, or chart version
- deployment method and relevant configuration with secrets removed
- PostgreSQL version, client driver, and operating system or Kubernetes version
- expected and observed behavior
- minimal reproduction steps and safe logs, metrics, or admin output

Search existing issues first. Keep one issue focused on one problem so it can be
reproduced and resolved independently.

## Security

Do not report vulnerabilities, credentials, private keys, or customer data in a
public issue. Follow [SECURITY.md](SECURITY.md) for private reporting.
